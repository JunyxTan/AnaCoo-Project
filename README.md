# AnaCoo Appointment Website

This project serves the AnaCoo appointment website. Appointment details are prepared in the form and sent to WhatsApp for manual confirmation.

## Install dependencies

```bash
npm install
```

## Configure environment

1. Copy `.env.example` to `.env`.
2. Set values:
   - `PORT`
   - `ALLOWED_ORIGIN` (for production domain)

## Run locally

```bash
npm start
```

Open: `http://localhost:3000`
